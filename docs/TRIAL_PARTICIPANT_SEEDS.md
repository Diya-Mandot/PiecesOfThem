# Trial Participant Seeds

This file tracks public Sanfilippo stories that are linked to actual clinical trial participation.

## Why separate these

- Trial-linked sources are much more sensitive than general advocacy stories.
- Some are clearly named public participants.
- Others are only available as anonymized or aggregate descriptions.
- The pipeline should preserve those distinctions instead of flattening everything into a single evidence pool.

## Confidence tiers

- `confirmed`: Public source clearly identifies the child and confirms participation.
- `likely-anonymized`: Public source clearly describes a treated child but does not identify them by name.
- `aggregate-only`: Public source confirms treated cohorts, not individual public narratives.

## Best anchors

- `Eliza O'Neill` for named UX111-linked comparison
- `Unnamed age-2 higher-dose UX111 child` for anonymized UX111 efficacy pattern
- `Ornella Aiach` for named non-UX111 gene-therapy comparison
- `Will Byers` for named non-gene-therapy trial comparison
