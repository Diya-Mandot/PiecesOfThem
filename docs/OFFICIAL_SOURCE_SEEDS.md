# Official Source Seeds

This file tracks low-risk official and institutional public sources that can be used directly for:

- FDA and PFDD framing
- natural-history context
- biomarker and clinical-benefit language
- trial-status and PDUFA context

## Included source classes

- FDA guidance pages
- ClinicalTrials.gov studies
- company press releases
- disease-foundation background pages

## Usage guidance

- Use these sources directly for context and evidence-package framing.
- Do not present them as patient fragments.
- Use them to inform schemas, milestone labels, comparison logic, and dashboard copy.
